# Streamlit 3D Visualization App

This project is a Streamlit web application designed for visualizing 3D objects. It allows users to interact with and explore 3D models in a user-friendly interface.

## Project Structure

```
streamlit-3d-viz-app
├── app.py             # Main application file for the Streamlit app
├── requirements.txt    # Lists the dependencies required for the project
└── README.md           # Documentation for the project
```

## Installation

To set up the project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   cd streamlit-3d-viz-app
   ```

2. Create a virtual environment (optional but recommended):
   ```
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Running the Application

To run the Streamlit app, use the following command:
```
streamlit run app.py
```

This will start the Streamlit server, and you can access the app in your web browser at `http://localhost:8501`.

## Usage

Once the app is running, you can upload your 3D object files and interact with them using the provided controls. The app supports various formats for 3D objects.
